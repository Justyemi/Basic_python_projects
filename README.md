# Simple Python Programs
A collection of beginner-friendly Python projects demonstrating fundamental programming concepts and basic data analysis techniques. Each project is contained in its own folder with clear, commented code.
## Projects Included
| Project | Description | Skills Demonstrated ||---------|-------------|---------------------|| **Weather Prediction** | Simple temperature prediction model using linear regression | Data loading, train/test split, model evaluation || **Linear Regression** | Basic linear regression implementation from scratch | Gradient descent, cost functions, prediction || **Bill Calculator** | Calculates individual shares with tax and tip | User input, arithmetic, formatting || **GPA Calculator** | Converts letter grades to GPA and calculates averages | Conditionals, loops, dictionary mapping |
